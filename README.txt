This Language Pack contains the German language files for Elgg 1.8.X (suitable release for "X" version of Elgg see Changelog) including the bundled plugins.

To install the language pack plugin simply copy the german_language_pack folder into the mod folder of your site and enable the plugin like any other plugin in the admin area of your site. If you had installed a previous release of this plugin, it's best to remove the language files that came with this release from the corresponding folders (or simply delete them together with the other Elgg files when upgrading your site to the latest release anyway). If there are any language files remaining in other plugin folders you should place this language pack plugin below the Elgg bundled plugins in your plugin list in the admin area to override these other language files.

The language used on your Elgg installation can either be set for the whole site via the admin area ("Configure" - "Settings" - "Basic Settings", or in German "Konfigurieren" - "Einstellungen" - "Grundeinstellungen" respectively) or each user can set the language individually on the user account's "Settings" page ("Einstellungen") via the Topbar link.


----------

Dieses Sprachpaket enthält die deutschen Sprachdateien für Elgg 1.8.X (passende Version für jeweiligen "X"-Release von Elgg siehe Changelog) inklusive der mitgelieferten Plugins.

Zum Installieren dieses Sprach-Plugins einfach den german_language_pack-Ordner in das mod-Verzeichnis Deiner Seite kopieren und das Plugin wie ein normales anderes Plugin im Adminbereich Deiner Seite aktivieren. Falls Du bereits einen älteren Release dieses Plugins installiert hattest, solltest Du die Sprachdateien, die in diesem Release enthalten waren, aus den entsprechenden Ordnern löschen (oder einfach mitsamt den anderen Dateien löschen, falls Du sowieso gerade auf die neueste Version updatest). Falls noch andere deutsche Sprachdateien in den Ordnern von anderen Plugins verblieben sind, solltest Du dieses Spach-Plugin unterhalb der bei Elgg mitgelieferten Plugins in der Pluginliste im Adminbereich verschieben, damit diese anderen Sprachdateien unberücksichtigt bleiben.

Die Festlegung der verwendeten Sprache kann entweder global für die gesamte Elgg-Installation über den Admin-Bereich ("Configure" - "Settings" - "Basic Settings" bzw. auf deutsch "Konfigurieren" - "Einstellungen" - "Grundeinstellungen") vorgenommen werden oder jeder Benutzer kann individuell die zu verwendende Sprache auf seiner Settings-Seite ("Einstellungen") via den Link in der Topbar festlegen.

----------

This Language Pack contains the German language files for Elgg 1.8.8 including the bundled plugins.

To install / update the language pack simply copy the files into the corresponding folders:

* the main language file into the "language" folder in your Elgg root folder,
* the plugins' language files into the corresponding "language"-subfolders in the "mod" folder,
* additionally, the language pack contains the installation language file that should be copied into the "install/language" folder and
* the JS language file that should go to "views/default/js/languages".

This German language pack retains Elgg's folder structure. Therefore, you can just copy all files/folders into your Elgg root folder.

The language used on your Elgg installation can either be set for the whole site via the admin area ("Configure" - "Settings" - "Basic Settings", or in German "Konfigurieren" - "Einstellungen" - "Grundeinstellungen" respectively) or each user can set the language individually on the user account's "Settings" page ("Einstellungen") via the Topbar link.


----------

Dieses Sprachpaket enthält die deutschen Sprachdateien für Elgg 1.8.8 inklusive der mitgelieferten Plugins.

Für die Installation / zum Updaten müssen die Dateien einfach in die entsprechenden Verzeichnisse kopiert werden:

* die Hauptsprachdatei in das "language"-Verzeichnis im Elgg-Rootverzeichnis,
* die Sprachdateien der Plugins in die entsprechenden "language"-Unterverzeichnisse der jeweiligen Plugins im "mod"-Verzeichnis
* zusätzlich gibt es die Installations-Sprachdatei, die in das "install/language"-Verzeichnis kopiert werden sollte und
* die JS-Sprachdatei, die in das "views/default/js/languages"-Verzeichnis gehört.

Diese Verzeichnisstruktur wurde in diesem Sprachpaket beibehalten. Es ist also möglich, die mitgelieferten Dateien/Verzeichnisse einfach in das Elgg-Rootverzeichnis zu kopieren.

Die Festlegung der verwendeten Sprache kann entweder global für die gesamte Elgg-Installation über den Admin-Bereich ("Configure" - "Settings" - "Basic Settings" bzw. auf deutsch "Konfigurieren" - "Einstellungen" - "Grundeinstellungen") vorgenommen werden oder jeder Benutzer kann individuell die zu verwendende Sprache auf seiner Settings-Seite ("Einstellungen") via den Link in der Topbar festlegen.
